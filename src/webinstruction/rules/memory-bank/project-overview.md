# GS KAYUMBU/TSS School Website - Project Memory Bank

## Project Overview
This is a comprehensive school website for GS KAYUMBU/TSS (Government School Kayumbu/Technical Secondary School) located in Rwanda. The website serves as the primary digital presence for the school, providing information about academics, admissions, student life, and administrative functions.

## School Information
- **Full Name**: GS KAYUMBU/TSS (Government School Kayumbu/Technical Secondary School)
- **Location**: Rwanda
- **Type**: Secondary school with technical and vocational programs
- **Student Population**: 800+ active students
- **Staff**: 50+ qualified teachers
- **Years of Operation**: 15+ years of excellence

## School Programs
1. **Ordinary Level (O-Level)**: Forms 1-3 (S1-S3)
2. **Advanced Level (A-Level)**: Forms 4-6 (S4-S6) with MCE combination
3. **Technical & Vocational (TSS)**: 
   - FAD (Fashion and Design)
   - BDC (Building and Construction)

## Key Features
- Student Portal with registration and enrollment
- Staff Portal for administrative functions
- Comprehensive admissions system
- Academic program information
- Student life and extracurricular activities
