# Design System & UI Components

## Visual Identity

### Color Scheme
- **Primary Colors**: School branding colors (to be documented from CSS)
- **Secondary Colors**: Complementary palette for accents
- **Neutral Colors**: Grays and whites for backgrounds and text
- **Status Colors**: Success, warning, error states for forms

### Typography
- **Primary Font**: System fonts with fallbacks
- **Headings**: Hierarchical sizing (H1-H6)
- **Body Text**: Readable font sizes with proper line height
- **Special Text**: Emphasis, captions, and labels

### Logo & Branding
- **School Logo**: school1.jpeg (circular crop, 100px dimensions)
- **Alternative Images**: school.jpeg for various sections
- **Placeholder Image**: usable.png for leadership team

## UI Components

### Navigation
- **Desktop Navigation**: Horizontal menu bar
- **Mobile Navigation**: Hamburger menu (☰ / ✕)
- **Active States**: Highlighted current page
- **Hover Effects**: Interactive feedback

### Buttons & CTAs
- **Primary Buttons**: Main actions (Apply Now, Explore Programs)
- **Secondary Buttons**: Supporting actions
- **Link Buttons**: Text-based navigation
- **Icon Buttons**: Social media and contact links

### Cards & Containers
- **Program Cards**: Academic program displays
- **Feature Cards**: Benefits and highlights
- **Info Cards**: Statistics and achievements
- **Timeline Cards**: Process steps and history

### Forms & Inputs
- **Text Inputs**: Standard form fields
- **Textareas**: Message and suggestion forms
- **Buttons**: Submit and action buttons
- **File Uploads**: Document submission (enrollment)

### Icons & Graphics
- **Font Awesome**: Version 6.4.2 for consistent iconography
- **Icon Categories**: 
  - Academic: fas fa-book, fas fa-graduation-cap
  - Technical: fas fa-tools, fas fa-laptop
  - Contact: fas fa-envelope, fas fa-phone
  - Social: fab fa-facebook, fab fa-twitter, etc.

## Layout Patterns

### Page Structure
- **Header**: Logo + Navigation
- **Hero Section**: Page title + description
- **Content Sections**: Modular content blocks
- **Footer**: Contact info + links + copyright

### Grid Systems
- **Responsive Grids**: Flexible column layouts
- **Card Grids**: 2-3 column layouts for features
- **Timeline Layout**: Vertical progression for processes

### Responsive Breakpoints
- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px

## Accessibility Standards
- **Color Contrast**: WCAG compliant ratios
- **Keyboard Navigation**: Tab-accessible elements
- **Screen Readers**: Semantic HTML structure
- **Alt Text**: Descriptive image alternatives
- **Focus States**: Visible focus indicators
